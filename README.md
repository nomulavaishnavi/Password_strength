# 🔐 Password Strength Checker (Python)

A simple **Password Strength Checker** built using **Python** and **Regular Expressions (regex)**.
This program checks whether a password is **strong or weak** based on common security rules.

---

## 🚀 Features

* 🔍 Checks password strength
* 🔢 Verifies presence of **numbers**
* 🔠 Checks for **uppercase letters**
* ✨ Detects **special characters**
* 📏 Ensures **minimum length requirement**
* ⚠ Displays missing requirements for weak passwords

---

## 🛠 Technologies Used

* **Python**
* **re module (Regular Expressions)**
* **Conditional Statements**
* **User Input Handling**

## ▶️ How to Run the Program

1. Install **Python 3** on your system.

2. Clone the repository:

```
git clone https://github.com/your-username/password-strength-checker.git
```

3. Navigate to the project folder:

```
cd password-strength-checker
```

4. Run the program:

```
python password_checker.py
```
